## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/uncle-phil.git
    ```
2. Navigate to the project directory:
    ```bash
    cd uncle-phil
    ```
3. Install:
    ```bash
    make install
    ```
